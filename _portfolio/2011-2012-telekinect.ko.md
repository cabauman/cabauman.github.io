_**2011 - 2012 DeVry GSP 졸업 프로젝트**_

저는 DeVry Austin 캠퍼스의 8인 졸업 프로젝트 팀 일원으로 PC용 게임 프로토타입을 개발했습니다. Unity3D 엔진 기반이며, Kinect를 활용해 자연스러운 사용자 인터페이스를 구현했고 멀티플레이(TCP/IP, 컴퓨터 2대 + Kinect 2대)도 지원했습니다. 개발 외에도 프롤로그 작성, 트레일러 제작, 게임 개발 동아리 웹사이트의 프로젝트 위키 구축을 담당했습니다.

Telekinect는 원소 기반 정신 감옥에서 탈출하려는 염동력 존재를 플레이하는 3인칭 액션 어드벤처 프로토타입입니다. 플레이어는 몸 동작으로 이동, 염동력, 원소 오브 투척을 조작합니다.

주요 기여
- 스폰 포인트
- "Fan challenge" 게임플레이
- 적 행동 로직
- 점프 메커닉
- 그림자 매핑 작업(가장 많은 시간 투입). 당시 Unity Basic에는 Render Texture 기능이 없어, 팀원과 함께 C++ DLL + OpenGL Framebuffer Object를 통해 우회 구현.

<blockquote><p>
원소 적의 웨이브를 돌파하고 물리/원소 기반 도전을 완료해 감금을 탈출하세요. 진행할수록 과거의 단서가 드러나며, 당신의 종족 Movers와 원래 평화롭던 Elementalists 사이 전쟁의 배경이 밝혀집니다.
</p></blockquote>

Kinect 게임의 흔한 불만은 "온레일(on-rails)" 구조로 인해 자유 탐험이 어렵다는 점인데, Telekinect는 이를 줄이려는 방향으로 설계했습니다.
