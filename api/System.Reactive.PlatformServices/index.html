System.Reactive.PlatformServices Namespace

Summary

The System.Reactive.PlatformServices namespace contains interfaces and classes used by the runtime infrastructure of Reactive Extensions. Those are not intended to be used directly from user code and are subject to change in future releases of the product.

EnlightenmentProvider Provides access to the platform enlightenments used by other Rx libraries to improve system performance and runtime efficiency. While Rx can run without platform enlightenments loaded, it's recommended to deploy the System.Reactive.PlatformServices assembly with your application and call EnsureLoaded() during application startup to ensure enlightenments are properly loaded.
